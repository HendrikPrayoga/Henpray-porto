Algorithm · Programmer

Optimasi Pengiriman Logistik E-Commerce

Proyek ini mengimplementasikan dan membandingkan tiga pendekatan algoritma — Dynamic Programming, Branch and Bound, serta Greedy — untuk menyelesaikan masalah optimasi rute pengiriman logistik pada platform e-commerce. Fokus utamanya adalah meminimalkan total biaya dan waktu distribusi dari gudang ke berbagai titik pengiriman secara efisien.

Sebagai Programmer, saya mengimplementasikan ketiga algoritma dalam Python, melakukan preprocessing data rute dan biaya menggunakan Excel, serta menganalisis perbandingan performa (kompleksitas waktu & ruang) antar algoritma untuk menentukan pendekatan optimal pada berbagai skala kasus.

  • Dynamic Programming untuk menyelesaikan subproblem overlapping pada rute multi-titik pengiriman.
  • Branch and Bound untuk eksplorasi solusi optimal dengan pemangkasan ruang pencarian.
  • Greedy heuristic sebagai solusi cepat berbasis keputusan lokal terbaik di setiap tahap.

Case Study · System Analyst

OrindPOS

Proyek ini merupakan pengembangan Sistem Point of Sale (POS) berbasis web untuk vape store yang mengintegrasikan pendekatan Software Engineering dan Data Science dalam satu solusi end-to-end. Dari sisi rekayasa perangkat lunak, sistem dirancang menggunakan arsitektur terstruktur (frontend–backend–database) dengan fitur manajemen transaksi real-time, pengelolaan stok, dashboard laporan penjualan, serta role-based access control untuk admin dan kasir.

Dari sisi data science, histori transaksi dimanfaatkan sebagai dataset untuk membangun model Machine Learning (Association Rule Learning & Collaborative Filtering) yang menganalisis pola pembelian dan menghadirkan rekomendasi produk otomatis demi mendukung strategi bundling dan peningkatan penjualan.

  • Integrasi workflow kasir–admin dengan logging real-time.
  • Pipeline ETL ringan yang menyiapkan dataset rekomendasi.
  • Evaluasi model berbasis lift chart dan uji A/B promosi.

Machine Learning · Project Manager

Prediksi Win/Loss Valorant

Proyek ini membahas implementasi dan evaluasi algoritma K-Nearest Neighbors (KNN) untuk memprediksi hasil pertandingan Valorant berdasarkan data statistik yang tersedia. Hasil pertandingan seperti kemenangan (Win) atau kekalahan (Loss) merupakan klasifikasi diskrit yang diprediksi dengan mempertimbangkan faktor-faktor yang memengaruhi performa pemain.

Sebagai Project Manager, saya mengkoordinasikan alur kerja mulai dari pengumpulan dan pembersihan data statistik pemain menggunakan Python dan Excel, eksperimen tuning parameter K, hingga evaluasi performa model dengan metrik akurasi, precision, dan recall.

  • Pipeline preprocessing data statistik pertandingan Valorant.
  • Tuning parameter K dengan cross-validation untuk akurasi optimal.
  • Evaluasi model menggunakan confusion matrix dan metrik klasifikasi.

Tugas Akhir · Lead Researcher

Pemetaan & Deteksi Kerawanan Bencana Tanah Longsor Menggunakan CNN dan Random Forest

Penelitian ini mengembangkan sistem deteksi dan pemetaan kerawanan bencana tanah longsor di Kecamatan Bumiayu, Kabupaten Brebes, dengan memanfaatkan dua algoritma machine learning sesuai karakteristik datanya: Convolutional Neural Network (CNN) untuk pemetaan berbasis data spasial raster, dan Random Forest untuk deteksi kerawanan berbasis data atribut spasial & temporal.

Sebagai Lead Researcher, saya bertanggung jawab penuh atas seluruh alur penelitian — mulai dari pengumpulan dan preprocessing data GIS menggunakan QGIS dan Excel, implementasi model di Python (Jupyter Notebook & Google Colab), hingga evaluasi dan komparasi performa kedua algoritma. Hasil penelitian diharapkan menjadi dasar sistem peringatan dini dan mitigasi bencana di wilayah rawan longsor.

  • CNN untuk segmentasi dan pemetaan zona kerawanan longsor berbasis citra raster multi-layer.
  • Random Forest untuk klasifikasi tingkat kerawanan berdasarkan faktor curah hujan, kemiringan lereng, jenis tanah, dan tutupan lahan.
  • Evaluasi komparatif kedua algoritma dengan metrik akurasi, precision, recall, dan F1-score untuk menentukan pendekatan terbaik sesuai fungsi analisisnya.
